gtt
===
This is a git-like package.

Usage
=====

Initializing a tgg repo:

```tgg init```

View commits and branches and current branch:

```tgg log```

Make a commit to the current branch:

```tgg commit -m "message"```

Create a new branch:

```tgg branch -b name```

Checkout to another branch:

```tgg checkout -b name```

Restore a commit:

```tgg revert -c commit_id```
